(i) What are the main benefits of using CSS?

Answer:

CSS allows for consistent styling, separation of content and presentation, and easier maintenance of web pages.

(ii) What are the different parts of a CSS style rule?

Answer:

A CSS style rule consists of a selector and a declaration block. The declaration block contains property-value pairs.

(iii) What is an element selector? Give an example.

Answer:

An element selector selects HTML elements based on their element name. For example, to select all

elements, you can use: p { color: blue; }

(iv) What is a class selector? Give an example.

Answer:

A class selector selects elements with a specific class attribute. For example, to select elements with the class "highlight," you can use: .highlight { background-color: yellow; }

(v) What is an id selector? Give an example.

Answer:

An id selector selects a single element with a specific id attribute. For example, to select an element with the id "main-heading," you can use: #main-heading { font-size: 24px; }

(vi) What are contextual selectors in CSS? Give an example.

Answer:

Contextual selectors select elements based on their relationship with other elements. For example, to select

elements within a

with the class "content," you can use: .content p { margin-left: 20px; }

(vii) What is an attribute selector? Give an example.

Answer:

An attribute selector selects elements based on the presence or value of an attribute. For example, to select all links with a "target" attribute, you can use: a[target] { text-decoration: underline; }

(viii) What is a pseudo selector? Give an example.

Answer:

A pseudo selector selects elements based on their state or position. For example, to select all unvisited links, you can use: a:link { color: blue; }

(ix) What are the three cascade principles used by browsers when style rules conflict?

Answer:

  1. Specificity: More specific selectors take precedence over less specific ones.
  2. Important: Rules marked with the "important" keyword have the highest priority.
  3. Order: Rules defined later in the stylesheet override conflicting rules defined earlier.
