Basic Concepts of Web Technology

Define HTML:

HTML stands for HyperText Markup Language. It is a standard markup language used to create web pages. HTML defines the structure and layout of web content using a set of tags and attributes.

Define Client-server model of communications:

The client-server model is a communication model where clients request services or resources from servers. Clients initiate requests, and servers provide responses. It is a fundamental concept in web technology.

Write down the name of different types of servers:

There are various types of servers, including web servers, email servers, file servers, DNS servers, and application servers, among others.

Define a Web Server:

A web server is a software and hardware system that stores and serves web content to clients. It responds to HTTP requests from browsers and delivers web pages, files, or other resources.

Define Data Centers:

Data centers are centralized facilities equipped with computing and networking infrastructure to store, manage, and process data and applications. They often house servers, storage devices, and networking equipment.

Why do we need a Uniform Resource Locator (URL)?

A Uniform Resource Locator (URL) is used to identify and locate resources on the web. It provides a standardized way to specify the address of web pages and resources, making them accessible to users.

Give an example of Uniform Resource Locator (URL) and specify all components of a URL:

Example URL: https://www.example.com:8080/path/to/resource?param=value#section
Components:

  • Scheme: https
  • Host: www.example.com
  • Port number: 8080
  • Path: /path/to/resource
  • Query: ?param=value
  • Fragment: #section

What is the difference between a static and a dynamic web site?

A static website contains fixed content that remains the same for all users. In contrast, a dynamic website generates content on the fly, often pulling data from databases or other sources, allowing for personalized and interactive experiences.

Why do we need Domain Name System (DNS)?

The Domain Name System/Service (DNS) is necessary to translate human-readable domain names (e.g., www.example.com) into IP addresses (e.g., 192.168.1.1) that computers can understand, facilitating internet communication.

Describe in detail the process of DNS address resolution:

The DNS address resolution process involves:

  1. The client queries a DNS resolver (often provided by the Internet Service Provider) with a domain name.
  2. The DNS resolver checks its cache for the corresponding IP address; if found, it returns the IP to the client.
  3. If not found, the resolver contacts root DNS servers, followed by top-level domain (TLD) and authoritative DNS servers, recursively, until it obtains the IP address.
  4. The resolver caches the result for future use.

Give some examples of generic top-Level domains (gTLD):

Examples of gTLDs include .com, .org, .net, .info, and .biz, among others.

Give some examples of country code top-Level domains (ccTLD):

Examples of ccTLDs include .us (United States), .uk (United Kingdom), .ca (Canada), .au (Australia), and .jp (Japan), among others.

What is the function (purpose) of Hypertext Transfer Protocol (HTTP)?

HTTP is a protocol used for transferring data between a web browser (client) and a web server. Its purpose is to enable the request and retrieval of web resources, such as web pages, images, and files.

Describe how a single web page is displayed on a web browser:

When a user enters a URL or clicks on a link, the browser sends an HTTP request to the web server. The server processes the request, retrieves the web page's HTML and associated resources (CSS, JavaScript, images), and sends them back to the browser. The browser then renders the page, applying styles and executing scripts to display the content.

Give some examples of HTTP request methods:

HTTP request methods include GET (retrieves data), POST (submits data), PUT (updates data), DELETE (deletes data), and more. They define the action to be performed on a resource.
